中文名：清除长期未使用的 Nuget 包

打开 RemoveLongUnusedNugetPackages，就会自动删除长期未使用的 Nuget 软件包。

目前只支持 Windows 操作系统，但在代码注释中有跨平台兼容的代码，但没有针对其他系统的 Nuget 软件包路径。

## ✨ 安装
